Border

Propiedad que describe el ancho, el estilo de linea y el color de los bordes de una caja. Este borde se puede apreciar en el contorno negro de toda la página.

  • Border-left
    Propiedad para poner el ancho, estilo y color del borde izquierdo de un elemento. Este borde se puede ver a la izquierda del contenedor, el contorno es azul.

  • Border-right
    Propiedad para poner el ancho, estilo y color del borde derecho de un elemento. Este borde se puede ver a la derecha del contenedor, el contorno es naranja.

  • Border-top
    Propiedad para poner el ancho, estilo y color del borde superior de un elemento. Este borde se puede ver en la parte superior del contenedor, el contorno es verde.

  • Border-bottom
    Propiedad para poner el ancho, estilo y color del borde inferior de un elemento. Este borde se puede ver en la parte inferior del contenedor, el contorno es amarillo.

  • Border-radius
    Propiedad que redondea las esquinas del borde exterior de un elemento. Es por esta propiedad que las esquinas de este contendor son redondas.

Color

El color es una propiedad que nos permite asociar un color a un elemento. En css podemos agregar colores por medio de palabras clave, hexadecimal o en el caso del color de este contenedor, rgb.
Este color es rgb(81, 189, 127).

Font

Propiedad que nos permite cambiar el estilo del texto así creando identidad para nuestro trabajo.

  • Font-family
    Dentro de CSS existen cinco familias genéricas: serif, sans-serif, monospace, cursive, fantasy. Estas se establecen en el html y se pueden importar otras fuentes.
    En este trabajo se ocupó la familia de monospace y se importo Shrikhand.

  • Font-style
    Propiedad para definir el aspecto de una familia tipográfica entre los siguientes tres valores: normal, italica, oblique.

    Propiedad especifica el peso o grueso de la letra. Existen normal y bold.

Tamaño

  • Width
    La propiedad se refiere a la anchura del área de contenido de un elemento. En este caso, el contenedor mide 500px de ancho.

  • Height
    La propiedad se refiere a la altura del área de contenido de un elemento. En este caso, el contenedor mide 370px de ancho.

    • ** Estos dos elementos al haber sido establecidos en el css se convierten en constantes por lo que no se modifican al tamaño de la página como lo hacen los otros contenedores.

List-style

Esta propiedad es un shorthand de List-style-type, list-style-position y list-style-image que permite asignar diferentes caractéres o imagenes a las viñetas de una lista. Cuando se utiliza, si alguno de los valores nombrado falta, se toma el valor por default.
  • Elemento 1, viñeta cuadrada
  • Elemento 2, viñeta con número romano en mayúscula

Margin

Establece el margen para los cuatro lados y es una breviación de margin-bottom/left/right/top.
margin: auto;
margin: 20px (arriba) 2px (derecha) 15px (abajo) 5px (izquierda);

Padding

Establece el espacio de relleno requerido por todos los lados de un elemento. El área de padding es el espacio entre el contenido del elemento y su borde.
** No acepta valores negativos
** Padding diferente a la linea anterior (el doble)

Overflow
