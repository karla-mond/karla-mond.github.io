Modificadores de Texto

<strong> </strong> - Strong

Elemento que enfatiza (aquí) las palabras más importantes de un texto.

<em> </em> - Em

Al igual que strong enfatiza (aquí) las palabras más importantes de un texto y puede ser anidado.

<i> </i> - i

Muestra el texto en itálicas

<b> </b> - b

Muestra el texto en negritas

<u> </u> - u

Muestra el texto en subrayado

La diferencia entre strong y b; y em e i es la semántica. Strong enfatiza semánticamente la palabra/palabras importantes, mientras que b es solo texto con un estilo convencional de negritas.

<stike> </strike> - Strike

Muestra el texto tachado con una línea horizontal, así.

<small> </small> - Small

Hace el tamaño del texto una talla más pequeña que el tamaño mínimo de fuente del navegador.

Texto normal:
Lorem ipsum dolor sit amet consectetur adipisicing elit. Esse, at?

Texto con <small> </small>:
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Perferendis, ab.

Dentro de section - <section> </section>

Elemento que representa una sección genérica de un documento, determina qué contenido corresponde a qué parte de un esquema. Está diseñado para contenidos dependientes, pero diferenciados. No se debe usar el elemento como un contenedor genérico.

Dentro del span - <span> </span>

Un span es un elemento que nos permite asignar cualquier atributo global de HTML a una sección de texto u otro contenido.

Dentro del article <article> </article>

Un article funciona como contenido independiente y autónomo. Algunos ejemplos de uso de este elemento son posts de forums, posts de blogs o historias de noticias.

Dentro de aside <aside> </aside>

Elemento que representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento. Normalmente consiste en barras laterales o incersiones y contienen una explicación al margen.

Listas

Lista sin orden

Lista de bullet points o viñetas no numeradas, se utiliza <ul> </ul>.

  • Elemento 1
  • Elemento 2

Lista ordenada

Lista numaerada, se utiliza <ol> </ol>.

  1. Elemento 1
  2. Elemento 2

Tabla

Para las tablas se establecen los headers de las tablas y después el cuerpo de ellas. <thead> es el elemento para nombrar las columnas, <td> funciona para establecer cada elemento de la tabla, la función de <tbody> es nombrar un bloque de filas y <tr> se refiere a una fila de la tabla.

Dentro del Footer - <footer> </footer>

Elemento que representa el pie de página para el contenido de sección más cercano. Usualmente contiene información acerca de el autor de la sección, datos de derechos de autor o enlaces a documentos relacionados.

Dentro del Nav - <nav> </nav>
